## 🎯 现在需要你完成3个简单步骤

### 步骤1：注册MongoDB Atlas（2分钟）

1. 打开浏览器，访问：https://www.mongodb.com/cloud/atlas/register
2. 点击"Sign Up"注册（可以用Google账号快速登录）
3. 创建免费集群：
   - 选择"Shared"（免费）
   - 区域选择：**Singapore (ap-southeast-1)**
   - 点击"Create"
4. 创建数据库用户：
   - 左侧菜单 → "Database Access"
   - 点击"Add New Database User"
   - 用户名：`surveyuser`
   - 密码：点击"Autogenerate Secure Password"并**复制保存**
   - 点击"Add User"
5. 设置网络访问：
   - 左侧菜单 → "Network Access"
   - 点击"Add IP Address"
   - 点击"Allow Access from Anywhere"
   - 点击"Confirm"
6. 获取连接字符串：
   - 左侧菜单 → "Database"
   - 点击"Connect"
   - 选择"Connect your application"
   - 复制连接字符串（类似下面格式）：
   ```
   mongodb+srv://surveyuser:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority
   ```
   - **重要：将 `<password>` 替换为你刚才保存的密码**

---

### 步骤2：部署到Vercel（2分钟）

#### 方法A：通过Vercel网站部署（最简单）

1. 访问：https://vercel.com
2. 点击"Sign Up"，选择"Continue with GitHub"
3. 授权Vercel访问GitHub
4. 点击"Add New..." → "Project"
5. 点击"Import Git Repository"
6. 选择"Import Third-Party Git Repository"
7. 输入GitHub仓库URL（如果你已上传）或选择"Upload"
8. 如果选择Upload：
   - 将整个项目文件夹压缩成ZIP
   - 上传ZIP文件
9. 配置项目：
   - Project Name: `survey-system`
   - Framework Preset: Other
   - Root Directory: `./`
10. 添加环境变量：
    - 点击"Environment Variables"
    - Name: `MONGODB_URI`
    - Value: 粘贴你的MongoDB连接字符串
    - 点击"Add"
11. 点击"Deploy"
12. 等待1-2分钟...
13. **完成！复制你的网站地址**

#### 方法B：通过GitHub自动部署（推荐）

1. 创建GitHub仓库：
   - 访问：https://github.com/new
   - 仓库名：`survey-system`
   - 选择"Public"
   - 点击"Create repository"

2. 推送代码（在项目文件夹执行）：
   ```bash
   git remote add origin https://github.com/你的用户名/survey-system.git
   git branch -M main
   git push -u origin main
   ```

3. 连接Vercel：
   - 访问 https://vercel.com
   - 点击"Import Project"
   - 选择你的GitHub仓库
   - 添加环境变量 `MONGODB_URI`
   - 点击"Deploy"

## 🆘 遇到问题？

### 问题1：Vercel部署失败

**解决方案：**
- 检查MongoDB连接字符串是否正确
- 确保环境变量名称是 `MONGODB_URI`
- 检查密码中是否有特殊字符需要URL编码

### 问题2：无法上传到Vercel

**解决方案：**
- 使用GitHub方式部署
- 或者使用Vercel CLI：
  ```bash
  npm install -g vercel
  vercel login
  vercel
  ```

### 问题3：数据不显示

**解决方案：**
- 打开浏览器F12开发者工具
- 查看Console是否有错误
- 检查Network标签，看API请求是否成功
- 确认MongoDB连接字符串正确

### 问题4：图表不显示

**解决方案：**
- 先提交几份测试问卷
- 刷新管理后台页面
- 检查浏览器Console是否有错误
